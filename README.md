# hb-industry-report

把行业访谈记录转化为面向行业小白（产品经理）的行业分析报告的 Claude Code skill。

**输入**：访谈纪要 / 录音转写 / 调研笔记（可选补充公开资料）
**输出**：单文件 HTML 报告——摘要 → 行业背景 → 商业模式 → 需求分析 → 术语表，内嵌 5 张 SVG 图（发展时间轴、产品服务全景、资金链双层图、L1 流程图、行业特色图），统一"藏青 × 铜金"设计系统，深浅色双主题。

## 特点

- **纯业务视角**：即使素材是围绕某个系统的演示访谈，也强制翻译回真实业务流程，全文不出现系统操作维度；
- **五段式业务块**：每个业务环节按"做什么 / 子流程 / 谁在参与 / 业务特征 / 痛点"展开；
- **数字纪律**：每个数字可溯源，单一案例、示意测算、公开资料推断均显式标注口径；
- **摘要可照着念**：三段式摘要，一分钟向别人讲清一个行业。

## 安装

```bash
git clone https://github.com/1145434396-boop/hb-industry-report.git ~/.claude/skills/hb-industry-report
```

## 使用

在 Claude Code 中提供访谈素材并说：

> 根据这份访谈，出一份行业分析报告
